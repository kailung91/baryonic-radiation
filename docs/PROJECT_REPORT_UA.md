# Звіт про розробку корпоративного сайту ПрАТ "Інститут передових технологій"

**Дата формування звіту:** 21.11.2025  
**Статус проекту:** Активна розробка (Фаза 1 завершена)

---

## 1. Технічна реалізація

### Технологічний стек
*   **Frontend Framework:** [Next.js 15.0.3](https://nextjs.org/) (App Router)
*   **Core Library:** React 19.0.0-rc
*   **Language:** TypeScript 5.x
*   **Styling:** [Tailwind CSS 3.4](https://tailwindcss.com/)
*   **Animations:** [Framer Motion 12.x](https://www.framer.com/motion/)
*   **Icons:** Lucide React
*   **UI Components:** Shadcn/ui (Radix UI primitives)
*   **Theme Management:** next-themes (Dark/Light mode)
*   **State Management:** Zustand (для складних станів), React Context

### Архітектура
*   **Тип:** Hybrid (SSR + CSR). Використовується Server Side Rendering для SEO-важливих сторінок та Client Components для інтерактивних елементів.
*   **Routing:** File-system based routing (Next.js App Router).
*   **Component Design:** Atomic Design principles (ui -> components -> features -> layouts).

### Інструменти
*   **Збірка:** Webpack (вбудований у Next.js).
*   **Linting:** ESLint + eslint-plugin-tailwindcss.
*   **Package Manager:** npm.

### Адаптивність та Продуктивність
*   **Responsive:** Mobile-first підхід, повна адаптивність для Mobile, Tablet, Desktop.
*   **Оптимізація:**
    *   Автоматична оптимізація шрифтів (`next/font`).
    *   Code splitting (автоматичний).
    *   GPU-accelerated анімації (transform, opacity).
    *   `useReducedMotion` для доступності.

## 2. Структура та функціонал

### Основні розділи
1.  **Головна (`/`)**: Hero-секція з паралаксом, картки особливостей, статистика.
2.  **Про нас (`/about`)**: Місія, візія, цінності, команда (замінив розділ GIS).
3.  **Продукти (`/products`)**: Каталог карт, атласів та додатків з фільтрацією та пошуком.
4.  **Освіта (`/education`)**: (В розробці) Освітні матеріали.
5.  **Юридичні сторінки**: Privacy Policy (`/privacy`), Terms (`/terms`).

### Функціонал
*   **Теми:** Перемикач світлої/темної теми з анімацією та збереженням вибору.
*   **Навігація:** Адаптивне меню (Desktop + Mobile Sheet).
*   **Каталог:** Клієнтська фільтрація та пошук продуктів у реальному часі.
*   **Інтерактивність:** Мікро-взаємодії на кнопках, полях вводу, картках.

### Адміністрування
*   На даному етапі контент керується через код (`src/lib/data.ts`).
*   Планується інтеграція Headless CMS (Sanity або Strapi) у наступних фазах.

## 3. UI/UX Дизайн

### Концепція
*   **Стиль:** "Premium Tech" / "Glassmorphism".
*   **Естетика:** Використання напівпрозорих поверхонь (glass), розмиття (blur), градієнтних акцентів та "шуму" (noise texture) для глибини.
*   **Кольори:**
    *   Primary: Signal Blue (`#0057B7`)
    *   Accent: Kinetic Gold (`#FFD700`) - кольори бренду/прапору.
    *   Background: Clean White / Deep Black.

### User Flow
*   Інтуїтивна навігація з чіткими CTA (Call to Action).
*   Плавні переходи між станами (hover, focus, active).
*   Зворотній зв'язок на дії користувача (тактильні анімації кнопок).

### Компоненти
*   `InteractiveButton`: Кнопки з ефектом натискання та ripple.
*   `FeatureCard` / `ProductCard`: Картки з ефектом підняття та світіння при наведенні.
*   `AnimatedInput`: Поля вводу з плаваючими мітками (Material style).

## 4. Контент

### Типи контенту
*   Текстові описи (місія, продукти).
*   Векторна графіка (іконки Lucide).
*   Метадані для продуктів (ціна, категорія, характеристики).

### Багатомовність
*   Поточна версія: **Українська/Англійська** (змішаний контент, базове налаштування `lang="uk"`).
*   Технічна готовність: Підготовлено структуру для i18n, але повний переклад ще не реалізовано.

## 5. SEO та Доступність

### SEO
*   **Metadata API:** Динамічні Title та Description для кожної сторінки.
*   **Semantic HTML:** Правильне використання тегів `<header>`, `<main>`, `<nav>`, `<h1>`-`<h6>`.
*   **Sitemap:** `sitemap.ts` генерує XML карту сайту автоматично.
*   **Robots:** `robots.ts` налаштований.

### Доступність (Accessibility)
*   **WCAG 2.1 AA:** Дотримання контрастності кольорів.
*   **Keyboard Navigation:** Повна підтримка навігації клавіатурою (Focus rings).
*   **Screen Readers:** ARIA-label атрибути для інтерактивних елементів.
*   **Reduced Motion:** Підтримка налаштувань системи для зменшення руху.
